== Description ==

Create categorical feedback assessment for your blog posts and allows your users to send their feedback.

Post Feedback is a plugin that allows your registered readers (member) to give feedback for your blog posts. The feedback is a categorical feedback, so you can specify any feedback option to evaluate your posts. The feedback results will be visualized with bar charts in your Admin Post page . In the settings page, you are able to specify color for each feedback option. At a glance, you will notice which post is amazing and which one that needs your attention.

It is a simple plugin but worth to install. It helps you to analyze performance and attraction level of your blog posts.

== Installation ==

1. Go to your WordPress dashboard, Plugins, and Add New Plugin.
2. Go to the 'Upload' section.
3. Upload 'post-feedback.zip' and activate the plugin.

== Frequently Asked Questions ==

= How many feedback options are allowed for the feedback? =

You can add up to 5 options for the feedback and specify color for each of them.
